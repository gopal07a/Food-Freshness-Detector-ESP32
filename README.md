# Food-Freshness-Detector-ESP32
ESP32 based food freshness detector using MQ135, DHT11, buzzer and web monitoring
# Food Freshness Detector (ESP32)

 Overview

This project detects food freshness using gas sensor MQ135 and displays data using a web interface.

 Components Used

* ESP32
* MQ135 Gas Sensor
* DHT11 Temperature & Humidity Sensor
* Buzzer
* LEDs
  
 Features
* Detects food freshness using gas levels
* Shows data on web browser (Wi-Fi)
* Alerts using buzzer
* Humidity calibration included

 Output

Open browser and enter ESP32 IP address to view:

* Gas Value
* Temperature
* Humidity
* Freshness Status

How to Run

1. Connect all components
2. Upload code to ESP32
3. Connect to Wi-Fi
4. Open IP address in Chrome
